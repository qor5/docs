2. 删除非当前选中也非当前显示
3. 选中其他项并删除，以测试删除当前选中是否会回选当前显示
4. 选中其他项但删除当前显示，测试其是否会切换显示但不更改选中
5. 删除当前显示并为当前选中，测试其是否会切换显示和选中为更老一个版本
6. 选中最老的版本并确认显示，然后再删除，以测试其在无更老版本切换的时候是否会切换和显示最新版本
